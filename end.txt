✅ CRUD Productos:
POST   /api/products          - Crear producto
GET    /api/products          - Listar productos
GET    /api/products/:id      - Obtener producto
PUT    /api/products/:id      - Actualizar producto
DELETE /api/products/:id      - Eliminar (desactivar) producto

✅ Punto de Venta:
POST   /api/sales             - Registrar nueva venta
GET    /api/sales             - Listar ventas (con filtros)

✅ Reportes:
GET    /api/reports/products-sold    - Productos vendidos por fecha
GET    /api/reports/top-products     - Top 3 productos más vendidos
GET    /api/reports/sales-chart      - Datos para gráfica de ventas
